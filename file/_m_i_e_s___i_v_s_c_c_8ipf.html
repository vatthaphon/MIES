Functions

variable IVS_ConfigureMCC()¶
variable IVS_runBaselineCheckQC()¶

Run the baseline QC check.

This will zero the amp using the pipette offset function call, and look at the baselineSSAvg, already calculated during the TestPulse. The EXTPINBATH wave will also be run as a way of making sure the baseline is recorded into the data set for post-experiment analysis

variable IVS_FinishBaselineQCCheck(WMBackgroundStruct *s)¶

Complete the Baseline QC check in the background.

variable IVS_runInitAccessResisQC()¶

Run the initial access resistance check from the WSE.

This will check must be < 20MOhm or 15% of the R input. The EXTPBREAKN wave will also be run as a way of making sure the reading is recorded into the data set for post-experiment analysis

variable IVS_finishInitAccessQCCheck(WMBackgroundStruct *s)¶

Complete the Baseline QC check in the background.

variable IVS_RunGigOhmSealQC()¶

run the GigOhm seal QC check from the WSE

This will make sure the Steady State resistance must be > 1.0 GOhm. The EXTPCIIATT wave will also be run as a way of making sure the baseline is recorded into the data set for post-experiment analysis.
