variable GetTraceColor(variable index, variable *red, variable *green, variable *blue)¶

Get distinctive trace colors for a given index.

Holds 21 different trace colors, code originally from http://www.igorexchange.com/node/6532 but completely rewritten and bug-fixed.

The colors are “Twenty two colors of maximum contrast” by L. Kelly, see http://www.iscc.org/pdf/PC54_1724_001.pdf, where the color white has been removed.

std::tuple<variable, variable> GetAxisRange(string graph, string axis, variable mode = defaultValue)¶

Query the axis minimum and maximum values.

For none existing graph or axis NaN is returned for minimum and high.

The return value for autoscale axis depends on the mode flag: AXIS_RANGE_INC_AUTOSCALED -> [0, 0] AXIS_RANGE_DEFAULT -> [NaN, NaN]

Return

minimum and maximum value of the axis range

Parameters

  • [in] graph: graph name

  • [in] axis: axis name

  • [in] mode: [optional:default AXIS_RANGE_DEFAULT] optional mode option, see AxisRangeModeConstants

wave GetAxesRanges(string graph = defaultValue, string axesRegexp = defaultValue, variable orientation = defaultValue, variable mode = defaultValue)¶

Returns a wave with the minimum and maximum values of each axis.

Use SetAxesRanges to set the minimum and maximum values

See

GetAxisRange

Return

free wave with rows = axes, cols = axes info, dimlabel of rows is axis name

Parameters

  • [in] graph: Name of graph

  • [in] axesRegexp: [optional: default not set] filter axes names list by this optional regular expression

  • [in] orientation: [optional: default not set] filter orientation of axes see AxisOrientationConstants

  • [in] mode: [optional: default AXIS_RANGE_DEFAULT] filter returned axis information by mode see AxisRangeModeConstants

variable SetAxesRanges(string graph, wave ranges = defaultValue, string axesRegexp = defaultValue, variable orientation = defaultValue, variable mode = defaultValue)¶

Set the range of all axes as stored by GetAxesRange.

Includes a heuristic if the name of the axis changed after GetAxesRange. The axis range is also restored if its index in the sorted axis list and its orientation is the same.

See

GetAxisRange

Parameters

  • [in] graph: Name of graph

  • [in] ranges: wave with graph ranges as set in GetAxesRanges

  • [in] axesRegexp: [optional: default not set] filter axes names list by this optional regular expression

  • [in] orientation: [optional: default not set] filter orientation of axes see AxisOrientationConstants

  • [in] mode: [optional: default 0] axis set mode see AxisRangeModeConstants
