string GetCompilationState()

Return the compilation state variable.

-NaN: Not initialized -0x0: Initialization done -0x1: input.txt could be read from disk and has contents -0x2: Compilation was started -0x4: Compilation was successfull -0x8: Compilation was not successfull (Requires Igor Pro 7, build 30097)

variable IsProcGlobalCompiled()

Return true if ProcGlobal is compiled, false if not.

Observed behaviour:

  • ProcGlobal is compiled after all independent modules

  • FunctionInfo returns an empty string for a non existing function, but an error message if ProcGlobal is not compiled
